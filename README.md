# resume
Resume built with Flask and Bootstrap

[resume](https://marcheseresume.herokuapp.com/)

## Getting Started

```
git clone https://github.com/joemarchese/resume
cd resume
python app.py
Navigate to http://localhost:5000/ on your favorite web browser.
```
