Tetratonic Squares

A music toy built with tkinter and pygame.

This is my first attempt at building something fun :D in Python using only tkinter and pygame. The word Tetratonic describes a scale with 4 notes. The menu allows the user to change the scale to a few sonorous presets or to whatever custom, dissonant combination their ear desires. The user can also choose their favorite MIDI instrument from (0) Acoustic Grand Piano - (127) Gunshot. I tried to give the instrument (or what imagined as a touch screen instrument) the most natural, logical setup where notes closest to the x-axis are lower in pitch and higher as you moved up or down and where notes closest to the y-axis are higher in velocity (volume) and decrease as you move further left or right. My hope is that the user would discover this organically through experimentation.


PolyNanna

A program designed to randomize a polyanna gift exchange with familial restrictions

This program attempted to solve a common problem at my family Christmas gathering; How to make a polyanna gift exchange fair and fun with extended family, preventing recipients in each participant's nuclear family. This presents a few good challenges: how to verify the drawing read: convince family members it actually works, how to hide results from the program operator, how to present the information to the participants, how to prevent repeats in subsequent years. I started building this program a few months after I started teaching myself to code. read: before I knew that I knew nothing. and the github commits reflect my learning process. From the first working commit with hardcoded data in the script, some unnecessary drawing logic, and poor organization. To the 2016 version with multiple classes, list comprehensions, history to prevent repeats, and data separated out into text files. To the Most Recent Version with a GUI version, a WebApp version built with Flask that utilizes a Database and Bootstrap, testing with unittest, a README, docstrings, set intersections and differences, and all the other capabilities of the language and libraries commonly used by Python developers.
